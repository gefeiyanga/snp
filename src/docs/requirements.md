# 个人资产与负债管理系统需求文档

## 1. 需求概述

### 1.1 产品名称
个人资产与负债管理系统

### 1.2 产品目标
帮助用户有效统计、管理和可视化其个人资产与负债情况，提供全面的财务概览，支持财务规划和决策。

## 2. 核心功能需求

### 2.1 资产管理功能

#### 2.1.1 资产分类
- **现金**: 包括手持现金、储蓄罐等
- **银行资金**: 各类银行账户余额（活期、定期存款等）
- **投资类资产**:
  - 虚拟货币（比特币、以太坊等）
  - 基金（股票型、债券型、混合型等）
  - 股票（A股、港股、美股等）
  - 黄金（实物黄金、纸黄金、黄金ETF等）
  - 其他金融投资产品
- **不动产**:
  - 房产（住宅、商业地产、土地等）
  - 车辆（汽车、摩托车等）
- **其他资产**:
  - 贵重物品（珠宝、收藏品、艺术品等）
  - 应收款项
  - 其他有价值资产

#### 2.1.2 资产记录功能
- 支持添加、编辑、删除各类资产
- 记录资产的基本信息（名称、类别、购买日期、当前价值等）
- 记录资产的历史变动情况
- 支持资产价值的实时或定期更新

### 2.2 负债管理功能

#### 2.2.1 负债分类
- **房贷**: 房屋贷款本金及剩余还款金额
- **车贷**: 车辆贷款本金及剩余还款金额
- **信用卡账单**: 各银行信用卡欠款
- **其他负债**:
  - 个人借款
  - 消费贷款
  - 其他金融负债

#### 2.2.2 负债记录功能
- 支持添加、编辑、删除各类负债
- 记录负债的基本信息（债权人、借款日期、利率、还款计划等）
- 跟踪负债的还款进度
- 支持负债利息计算
- 支持分期还款计划管理

#### 2.2.3 负债月度统计功能
- 支持按月统计负债偿还情况
- 显示每月需要偿还的金额
- 跟踪每月还款进度
- 提供月度负债汇总报表

### 2.3 统计与展示功能

#### 2.3.1 资产负债概览
- 实时显示总资产、总负债和净资产
- 以图表形式展示资产与负债分布
- 提供净资产值的变化趋势图

#### 2.3.2 分类统计
- 按资产类别展示占比和金额
- 按负债类别展示占比和金额
- 提供详细的分类统计数据

#### 2.3.3 月度负债统计
- 按月份展示负债偿还情况
- 显示每月负债总额变化趋势
- 展示各月的负债构成分析

#### 2.3.4 数据可视化
- 饼图展示各类资产/负债占比
- 折线图展示资产/负债变化趋势
- 柱状图对比不同类别资产/负债金额
- 柱状图展示月度负债还款情况

## 3. 非功能性需求

### 3.1 用户体验
- 界面简洁直观，易于操作
- 响应式设计，适配移动端和桌面端
- 支持快速录入资产和负债信息

### 3.2 数据安全
- 用户数据加密存储
- 支持本地数据备份
- 可选择数据同步到云端（如有云服务）

### 3.3 性能要求
- 页面加载快速
- 数据查询响应迅速
- 图表渲染流畅

## 4. 技术实现考虑

### 4.1 前端框架
- 基于当前项目技术栈（Vue 3 + TypeScript + Less + PWA + Vant + LocalForage）

### 4.2 数据存储
- 使用LocalForage进行持久化数据存储
- 实现数据模型以管理资产和负债信息

### 4.3 UI组件
- 利用Vant组件库构建移动端友好的界面
- 使用图表库展示数据可视化

---

**备注**: 以上需求为基础功能框架，具体细节可在后续开发过程中进一步完善和细化。
